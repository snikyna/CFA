# CFA Level I Study Materials

This repository contains comprehensive study materials for the CFA Level I examination.

## Structure

### Core Topics Covered:
- **Financial Analysis**: Statement analysis frameworks and methodologies
- **Economics**: Microeconomic and macroeconomic principles  
- **Quantitative Methods**: Mathematical foundations and statistical analysis
- **Corporate Issuers**: Organizational structures and governance

### Study Materials:
- LaTeX formatted notes for easy compilation and formatting
- Interactive HTML exhibits for visual learning
- PDF versions of compiled notes
- Supporting images and diagrams

### Usage:
1. Review LaTeX source files for detailed content
2. Compile to PDF for reading and printing
3. Use interactive exhibits for concept visualization
4. Cross-reference with official CFA curriculum
